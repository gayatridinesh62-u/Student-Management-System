# Student-Management-System
## 📌 Project Description
This is a Student Management System developed using Python Flask, HTML, CSS, and MySQL.

The system allows users to:
- Add student details
- View student records
- Login authentication
- Manage student data efficiently

## 🛠 Technologies Used
- Python (Flask)
- HTML
- CSS
- MySQL

## 🚀 Features
- User Login System
- Add Student Page
- View Student Page
- Dashboard
- Clean and Responsive Design

## 🎯 Purpose
This project is created for learning full-stack web development and database integration.
